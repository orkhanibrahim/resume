# Resume Builder

This project demonstrates a simple Spring Boot application using Thymeleaf.
Users can dynamically add or remove resume sections and entries in the form.

## Building

Run Maven tests and package the application:

```bash
mvn test
mvn package
```

Launch the application with `java -jar target/resume-0.0.1-SNAPSHOT.jar` and open [http://localhost:8080/resume](http://localhost:8080/resume).
